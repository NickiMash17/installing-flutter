Complete Flutter Installation Guide for Beginners

Step-by-Step Instructions ‚Ä¢ Created by Nicolette Mashaba

Before We Start: What is Flutter?

Flutter is a free and open-source toolkit from Google for building beautiful, native-compiled applications for mobile (iOS & Android), web, and desktop from a single codebase. You write your code once in the Dart language, and Flutter builds it for all these different platforms.

Step 1: Check Your System Requirements

For All Platforms:

  • Disk Space: At least 2-3 GB of free space (more for IDEs and SDKs)
  • Tools: You will need a command-line tool (Command Prompt, Terminal, etc.)

For Windows:

  • OS: Windows 10 or later (64-bit)
  • PowerShell 5.0 or newer (this comes pre-installed on Windows 10/11)
  • Git for Windows (we will install this in the steps)

For macOS:

  • OS: macOS version 10.14 (Mojave) or later
  • Tools: You will need to install Xcode for iOS development (optional if you only want to develop for Android)

For Linux (Ubuntu/Debian):

  • OS: 64-bit version
  • Libraries: You need a few command-line tools which we will install

Step 2: Install Git

Git is a version control system that Flutter uses behind the scenes.

  • Windows: Go to git-scm.com. Download and run the installer. Use all the default settings.
  • macOS & Linux: Git is often pre-installed. To check, open your terminal and type git --version. If it's not installed, it will prompt you to install it.

Step 3: Download and Install Flutter SDK

This is the core Flutter software.

Download the SDK:

  1. Go to the official Flutter website: https://docs.flutter.dev/get-started/install
  2. Click the button to download the Flutter SDK for your operating system (Windows, macOS, or Linux)

Extract the File:

  • Windows: Extract the downloaded .zip file to a location you prefer. A good place is C:\src\flutter (you might need to create the src folder).
  • macOS/Linux: Extract the file to your desired location. A good place is your home directory. For example, you can extract it to /Users/<your_username>/flutter on macOS or /home/<your_username>/flutter on Linux.
‚ö†Ô∏è Important: Do not extract Flutter to a folder that requires special permissions (like C:\Program Files\ on Windows).

Step 4: Add Flutter to Your Path

This tells your computer where to find the Flutter commands.

For Windows:

  1. Press the Windows key, type "env", and select "Edit the system environment variables"
  2. Click the "Environment Variables..." button at the bottom
  3. In the lower "System variables" section, find and select the Path variable, then click "Edit..."
  4. Click "New" and add the full path to the flutter\bin folder. For example: C:\src\flutter\bin
  5. Click OK to close all the windows

For macOS & Linux:

  1. Open your Terminal
  2. Determine which shell you are using. Run echo $SHELL
    • If it says /bin/zsh (common on newer macOS), you need to edit ~/.zshrc
    • If it says /bin/bash, you need to edit ~/.bashrc or ~/.bash_profile
  3. Open the file in a text editor. For example, for ZSH:
    nano ~/.zshrc
  4. At the end of the file, add this line (replace /path/to/ with your actual path):
    export PATH="$PATH:/path/to/flutter/bin"
    • Example for macOS: export PATH="$PATH:/Users/your_username/flutter/bin"
    • Example for Linux: export PATH="$PATH:/home/your_username/flutter/bin"
  5. Save the file (In nano: Press Ctrl+X, then Y, then Enter)
  6. Run this command in the terminal to reload the file, or simply close and reopen the terminal:
    source ~/.zshrc # or source ~/.bashrc

Step 5: Run Flutter Doctor

This is the most important command! It checks your system and tells you what's installed and what's missing.

  1. Open a new command prompt or terminal window (this is crucial so it picks up the new PATH)
  2. Type the following command and press Enter:
    flutter doctor
  3. The tool will run a diagnostic and show you a report

‚úÖ Checkmarks mean everything is good.
‚ùå Crosses/X marks mean something is missing.
‚ùó Exclamation marks mean something is installed but needs configuration.

Step 6: Install the Missing Dependencies (Based on flutter doctor)

Follow the instructions from the flutter doctor output.

1. Install the Android Toolchain (for Android development):

  • Install Android Studio: Go to developer.android.com/studio and download/install it
  • During installation: The wizard will ask which packages to install. Make sure Android SDK, Android SDK Platform, and Android Virtual Device are selected
  • Launch Android Studio: Once it opens, go through the setup wizard. It will install the latest Android SDK and other required components
  • Accept Android Licenses: Go back to your terminal/command prompt and run:
    flutter doctor --android-licenses
    Press y for every license prompt

2. Install Xcode (for iOS development on macOS only):

  • Open the App Store on your Mac
  • Search for "Xcode" and install it. This is a very large download (several GB)
  • After installation, open it once to agree to the license and complete the installation of additional components

3. Install an Editor (Highly Recommended):

You can use any text editor, but an IDE (Integrated Development Environment) will make your life much easier with features like code completion, debugging, and emulator management.

Android Studio / IntelliJ IDEA:

  1. Open Android Studio/IntelliJ
  2. Go to Preferences/Settings > Plugins
  3. Search for "Flutter", install it, and it will prompt you to install the Dart plugin as well
  4. Restart the IDE

Visual Studio Code (VS Code):

  1. Download and install VS Code from code.visualstudio.com
  2. Open VS Code
  3. Go to the Extensions tab (Ctrl+Shift+X / Cmd+Shift+X)
  4. Search for and install the "Flutter" extension

Step 7: Set Up an Emulator/Simulator (To Run Your Apps)

You need a device to test on. You can use a virtual one on your computer.

For Android (Android Emulator):

  1. Open Android Studio
  2. Go to Tools > Device Manager
  3. Click "Create Device"
  4. Select a phone definition (e.g., Pixel 5) and click Next
  5. Download a system image (e.g., Android API 34, Tiramisu) and click Next
  6. Finish the setup and click the Play button to start the emulator

For iOS (iOS Simulator - macOS only):

  1. Open Xcode
  2. Go to Xcode > Preferences > Components to install a simulator version if needed
  3. Open your terminal and run:
    open -a Simulator
    This will launch the iOS simulator

Step 8: Create and Run Your First Flutter App

You're all set! Let's test it.

  1. In your terminal, navigate to where you want to create your project and run:
    flutter create my_first_app
    (This creates a new folder called my_first_app with a simple demo app inside.)
  2. Navigate into the project folder:
    cd my_first_app
  3. Ensure your emulator is running or a physical phone is connected via USB
  4. Run the app:
    flutter run

After a moment, you should see the default Flutter counter app running on your emulator/device. Congratulations, you have successfully installed Flutter! üéâ
